CephEWS Help

Overview

Provides a overview of whole cluster status. From the top to the bottom, user can see the cluster health, space usage, node health, historical graph(number of objects, number of requests, etc.) for each pool.

Topology

Presents the current topology of the whole cluster in a tree graph. When admin clicks on a particular OSD, a popover window will show the detailed information about that OSD and its host. If collectd is enabled, user will be able to see the heat-map of OSD, showing the requests each OSD handled in the past period of time.

Pool

Presents a list of pools in the cluster. User can create or delete pools.

CRUSH Rule

Presents a list of CRUSH Rule, which are used to control objects distribution. User should be able to create and delete rules.

Object Distribution

Given pool name and ID of a particular volume, this tab will present the distribution of objects belonging to that volume. Distribution will be shown in terms of placement groups and OSDs.
